A menu bar icon that's actually Goji, and builds anyone can install.

## What's new

- The menu bar icon is now the goji berry mark instead of a generic microphone: outline while idle, filled while recording. The waveform and download indicators stay for transient states.
- Releases are now signed with a Developer ID certificate and notarized by Apple, so downloads open first try with no Gatekeeper warnings.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to your Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
