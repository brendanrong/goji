Updates install themselves now.

## What's new

- One-click updates: "Update to Goji X…" in the menu bar (or Install Update in About) downloads the new version, installs it, and relaunches Goji. No browser, no DMG, no dragging. If Goji isn't running from Applications it falls back to the manual download.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to the Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
