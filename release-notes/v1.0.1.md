A friendlier first run, and the Dock icon now does something.

## What's new

- Welcome window on first launch: fresh installs now get an explicit choice before the one-time ~600 MB speech model download, with a live progress bar and phase labels. Close the window and the download keeps going, with progress in the menu bar. Already have the model (or a bundled build)? You'll never see it.
- Clicking Goji's Dock icon now opens Settings (or the welcome window if the model still needs downloading) instead of doing nothing.
- The menu bar shows download state and offers the download and retry actions directly.

## Fixes

- Long captions in the welcome window wrap instead of truncating.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to your Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
