Tidier About, and a hint for stuck permissions.

## What's new

- About, tightened: everything lines up left like the rest of Settings, no more floating in space.
- If pasting ever stops while the Accessibility toggle looks on, the menu now tells you the fix: remove Goji from the list and add it back.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to the Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
