A clearer Transcription pane, and smarter name handling.

## What's new

- The Transcription pane reads in pipeline order (your words, cleanup, formatting, replacements), and each section says whether it needs Apple Intelligence. Only AI cleanup does.
- AI cleanup handles multiple names properly: a word that already matches one of your Names & phrases is never rewritten, and corrections always pick the closest-sounding term instead of collapsing everything to the first name in the list.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to the Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
