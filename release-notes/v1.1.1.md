Smoother downloads, cleaner installer.

## What's new

- The model download progress bar no longer stutters: progress reports were flooding the UI hundreds of times a second, now they arrive in tidy half-percent steps. Same fix applies to in-app updates.
- The installer window got a redesign: flat cream background, minimal arrow, bigger icons, no clutter.
- The repo has a proper README now.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to the Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
