The notch HUD now looks the same on every display.

## What's new

- Consistent notch HUD on external monitors: displays without a real notch get a synthetic notch island at the top of the screen, same design as on the MacBook, instead of a floating pill.
- The DMG now includes an Applications shortcut, so installing is the standard drag-and-drop.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to the Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
