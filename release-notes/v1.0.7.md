Make Goji yours: light, dark, or system.

## What's new

- Appearance setting: follow the system, or keep Goji light or dark. Settings > General > System, applies instantly.
- Settings controls are now sized like System Settings instead of the chunky defaults.
- Much faster first-run model download: one file from a fast CDN with real progress, instead of a 23-file crawl that could stall for minutes. Falls back to the old path automatically if needed.
- "Mute other audio" is now "Pause media while dictating": it pauses whatever's playing (Music, Spotify, browser video) and resumes it after, which works on every output device, including monitors that expose no volume control.
- Fixed the harsh sidebar contrast in light mode.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to the Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
