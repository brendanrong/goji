Hands-free dictation and faster mic switching.

## What's new

- Double-tap to lock: in Hold mode, tap the hotkey twice quickly and recording stays on hands-free. Tap once more to finish. Toggle it under Settings > General if you prefer plain push-to-talk.
- Microphone picker in the menu bar: see which mic Goji will use and switch it without opening Settings. The list refreshes every time you open the menu, so a freshly plugged-in mic shows up immediately.
- As always, the audio engine rebuilds per recording, so device changes (AirPods connecting, USB mics) are picked up automatically when you use System Default.

## Install

1. Download Goji.dmg below.
2. Open it and drag Goji to your Applications folder.
3. Launch Goji. Grant Microphone (prompt) and Accessibility (System Settings > Privacy & Security > Accessibility).

Requires macOS 14 or later. Apple Silicon recommended.
